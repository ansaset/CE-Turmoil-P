# CE-Turmoil-P

这是什么项目？

-这是Turmoil游戏的外挂，用于研究探讨。

为什么创建项目？

-记录练习日志，我会的不是很多。

如何使用？

-打开ce，添加游戏和ce文件即可。


      实现功能：
      替换油价
      加快抽油（时间变速）
      冻结时间0-27000

      待实现功能：

      左家和右家实时信息

      更改最低油价

      当前lvl金额初始值2000

      显示 石油大亨（钱，钻石，股份，其他AI财产）

      显示 热力沸腾（钱，收藏，股份，其他AI财产）
